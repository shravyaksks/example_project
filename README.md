Simple HTML & CSS Login Page
A clean, modern, and responsive login interface built using pure HTML5 and CSS3. 
This project features a centered "card" layout with custom focus effects and a professional color palette.

🚀 Features
 * Pure HTML/CSS: No external libraries or frameworks required.
 * Fully Responsive: Uses Flexbox for perfect centering on any screen size.
 * Interactive UI: Includes hover transitions and focus states for input fields.
 * Ready to Use: Can be easily integrated into any web application or CMS.
📂 Project Structure
├── index.html  
└── README.md    

🛠️ Installation & Usage
Since this is a static project, you don't need to install any dependencies.
 * Clone or Download the repository to your local machine.
 * Open in VS Code: Navigate to the folder and open it.
 * Run the project:
   * Option 1: Right-click index.html and select "Open with Live Server" (recommended).
   * Option 2: Simply double-click index.html to open it in your default web browser.
